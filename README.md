# cc-claude-code

`cc-claude-code` 是一个面向终端场景的 AI 学习与实践项目，聚焦于命令行交互、智能工具调用和工程化能力训练。

## 项目简介

该项目包含了从命令入口、会话管理、工具编排到权限控制、插件系统、MCP 集成等关键能力，适合用于：

- 学习 AI 工具型应用的整体架构设计
- 练习命令系统、工具系统和交互流程的实现方式
- 作为个人或团队的 AI 工程实验项目进行扩展

## 主要模块

- `src/entrypoints/`：程序入口与启动流程
- `src/commands/`：命令定义与分发逻辑
- `src/components/`：终端 UI 组件（React/Ink）
- `src/services/`：业务服务层（会话、权限、扩展能力等）
- `src/tools/`：工具定义、调度与执行相关逻辑
- `src/utils/`：通用工具函数与基础设施能力
- `vendor/`：配套模块或底层能力源码
